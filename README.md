# Фестиваль возможностей

- в стилях прописать `root`. Определить основную палитру цветом, шрифты и т.д. как переменные css
- сбросить стили через селектор *
- сначала определить селекторы
- потом классы, все стили делать через именнованные классы
- сформировать компонент `Header`. Настроить лого и меню
- компонент компетенции `Competences`
- компонент участников `Users`
- компонент регистрации `Sign`
- `npm install react-router-dom`
- настроить маршруты в `App` 
- компонент `Slider` (добавить несколько изображений, `setInterval`)
- блок описания концепции и основных ценностей
- список анонсов мероприятий (создать карточку мероприятия и вывести списком, варьировать через параметры компонента)
- блок информации об организаторах и команде
- блок счетчика дней до фестиваля

- компонент `Footer` (2 grid: основные разделы приложения, контакты организаторов)

- компонент `Compenetces` (карточка компетенции с выводом из API, в карточке возможность скачать задание, сделать спойлер в карточке)

- компонент `Users` (карточка пользователя с фото, вывод из API, поиск по компетенции, региону, категории)

- компонент `Sign` (форма с полями, ввод фото, валидация, список регионов подгружается с API)

- компонент `Auth`(логин, пароль)

- компонент `Profile`(список своих заявок, поиск компетенции, можно сформировать заявку)
- 
- компонент `AdminPanel`(создать компетенцию, загрузка файла с заданием)
